# The Classroom

The **member-only** classroom — the actual teaching, written out section by section. Members reach it via the **Classroom** tab in the sidebar (`/classroom`); it is not public-facing.

This folder is the markdown mirror. The **live content lives in `app/(member)/classroom/content.tsx`** (the `SECTIONS` array) — that is the source of truth the app renders. Keep this doc in sync at a high level; don't duplicate full lesson text here.

Voice is Javi's. When editing, preserve the prose as written — organize and structure, don't rewrite.

## How a section is laid out

Section One sets the pattern: a one-breath summary up top, then numbered **Modules**, ending with a "What's next" that hands off to the following section. Each section closes the loop on intuition — understand the levers, build the habit, troubleshoot when it drifts.
